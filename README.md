## 🚀 Tecnologias Utilizadas

**Estrutura**:  HTML, CSS. Base e estilo do site. <br>
**Interação**: JavaScript, SwiperJS. Funcionalidades dinâmicas e carrossel de projetos. <br>

**Ícones**: Unicons. Ícones modernos e responsivos.
